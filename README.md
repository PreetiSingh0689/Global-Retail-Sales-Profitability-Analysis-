# Global-Retail-Sales-Profitability-Analysis-

# 📌 Project Overview
This project delivers a comprehensive data analytics solution designed to evaluate, optimize, and streamline supply chain operations, freight costs, and profit margins for a global retail enterprise. By moving away from raw database fields and structuring executive-level reporting, this dashboard empowers operations managers to identify cost leaks and maximize regional profitability.

# 🎯 Business Objectives
Logistics Performance Analysis: Evaluate total order volumes, throughput, and operational efficiency across distinct shipping and distribution channels.
Freight Cost Optimization:Isolate systemic cost leaks by identifying regions and product sub-categories with disproportionately high average shipping expenses.
Profitability Mapping:Correlate financial performance (Sales vs. Net Profit) against supply chain metrics to uncover bottom-line constraints.
Customer Sentiment Tracking:Analyze textual customer feedback to uncover how delivery speed, delays, and product quality issues directly influence revenue decreases.


#⚙️ Analytical Methodology
1. Data Modeling & Schema Design: Engineered a clean, corporate star schema by establishing relationships between Dimension and Fact table.
2. Data normalization : normalize one large single table into dimension table.
3. Exploratory Data Analysis (EDA):Conducted deep-dive statistical analysis to investigate the relationship between product dimensions, delivery regions, and freight expenses.
4. Advanced Analytics & Root Cause Identification:Built an AI-driven root-cause framework to analyze text-based feedback and uncover hidden operational barriers impacting corporate profitability.
5. UX/UI & Dashboard Architecture:Created an executive-grade interface utilizing a cohesive, professional monochrome blue palette, web-app style button navigation, dynamic tooltips, and unified global currency formatting.


## 🛠️ Tools & Technologies Used
